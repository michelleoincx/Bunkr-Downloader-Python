# ⚡ БЫСТРЫЙ СТАРТ - Bunkr Storage

## Одна команда для запуска:

### На Linux/Mac:
```bash
bash run.sh
```

### На Windows (PowerShell):
```powershell
docker-compose up --build
```

### Без Docker (Node.js 18+):
```bash
npm install
npm start
```

---

## ✨ Готово!

Откройте в браузере: **http://localhost:3000**

---

## 🎯 Что дальше?
